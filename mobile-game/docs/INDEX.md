## Documentation Standards

All documentation follows these standards:

- **Format:** Markdown with GitHub Flavored Markdown extensions
- **Diagrams:** Mermaid for flowcharts, sequence diagrams, timelines
- **Code Examples:** Syntax-highlighted, executable
- **Structure:** Table of contents, clear headings, consistent formatting
- **Links:** Cross-referenced, absolute paths for robustness
- **Maintenance:** Reviewed and updated quarterly
