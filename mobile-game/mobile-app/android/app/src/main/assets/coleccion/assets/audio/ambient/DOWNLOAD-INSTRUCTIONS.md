# Instrucciones para Descargar Audios Ambientales

Para que la música ambiental funcione, necesitas descargar 12 archivos MP3 gratuitos y colocarlos en esta carpeta.

## Fuentes Recomendadas (CC0 - Uso Libre)

- **Pixabay**: https://pixabay.com/sound-effects/
- **BigSoundBank**: https://bigsoundbank.com/
- **Zapsplat**: https://www.zapsplat.com/ (requiere registro gratuito)
- **Orange Free Sounds**: https://orangefreesounds.com/

## Archivos Necesarios

Descarga un archivo para cada ambiente y renómbralo exactamente como se indica:

1. **rain.mp3** - Lluvia suave (30-60 segundos, loop)
2. **ocean.mp3** - Olas del mar
3. **forest.mp3** - Bosque con pájaros
4. **river.mp3** - Río fluyendo
5. **cafe.mp3** - Ambiente de cafetería
6. **fire.mp3** - Fuego/chimenea
7. **storm.mp3** - Tormenta con truenos
8. **wind.mp3** - Viento suave
9. **night.mp3** - Sonidos nocturnos
10. **birds.mp3** - Canto de pájaros
11. **meditation.mp3** - Cuencos tibetanos o música meditativa
12. **piano.mp3** - Piano ambiental suave

## Ejemplo con Pixabay

1. Ve a https://pixabay.com/sound-effects/search/rain/
2. Busca "rain" o el ambiente que necesites
3. Haz clic en "Download" (no requiere atribución)
4. Renombra el archivo a `rain.mp3`
5. Colócalo en esta carpeta: `www/assets/audio/ambient/`

## Verificar Instalación

Después de descargar todos los archivos, verifica que estén todos:

```bash
ls -1 www/assets/audio/ambient/*.mp3
```

Deberías ver 12 archivos .mp3

## Licencias

Todos los archivos deben ser:
- **CC0** (dominio público) o
- **Royalty-free** con permiso para uso comercial

Las fuentes recomendadas arriba cumplen estos requisitos.
